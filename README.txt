# Library

Es un proyecto de una libreria con la finalidad de poder cargar, editar y eliminar un autor, al igual que una editorial y un libro. Tambien para registrarse 
y tener diferentes vistas por roles (Usuario y Admin). El Admin puede tener acceso a lo mencionado anteriormente mientras que el usuario (cliente) puede registrarse 
y solicitar el prestamo de un libro. Sigue en constante cambios este proyecto para lograr optimizarlo al maximo.
